# Portfolio Backend - Marine

Ce projet est la partie serveur (API) de mon portfolio. Il est construit avec **Node.js**, **Express** et **TypeScript**.

## 🚀 Fonctionnalités
- **Contact Form** : Gestion de l'envoi d'e-mails via Nodemailer.
- **Sécurité** : Utilisation de variables d'environnement (`dotenv`) et gestion des CORS.
- **Qualité** : Tests unitaires et d'intégration avec Jest et Supertest.

## 🛠️ Technologies
- **Backend** : Node.js, Express
- **Langage** : TypeScript
- **Tests** : Jest, Supertest
- **Email** : Nodemailer
